# Transcriptome analysis (fungi)

This repository contains the main HTML reports and selected scripts/results for:
- QC (MultiQC; Salmon quantification)
- Differential expression (DESeq2)
- Enrichment analysis (EggNOG-mapper; fungi-filtered)
- InterProScan domain/signature analysis + GO-based biological categorization (e.g., DDR, transport, ETC, metal tolerance)

## Contents
- 01_qc/
- 02_differential_expression/
- 03_enrichment/
- 04_interproscan/

## Notes
Large raw data and intermediate outputs (reads, indices, caches) are excluded via .gitignore.
